# JLPT N5 Kanji Readings

## Introduction

JLPT N5 requires knowledge of approximately 80-100 basic kanji characters. This guide covers essential kanji with their on'yomi (音読み - Chinese reading) and kun'yomi (訓読み - Japanese reading), along with common example words to help you learn and practice.

---

## Understanding On'yomi and Kun'yomi

### On'yomi (音読み)
- Chinese-based pronunciation
- Usually used in compound words (words with 2+ kanji)
- Often written in katakana in dictionaries

### Kun'yomi (訓読み)
- Native Japanese pronunciation
- Usually used when kanji stands alone or with hiragana
- Often written in hiragana in dictionaries

## Learning Tips for N5 Kanji

1. **Learn radicals**: Understanding radicals (kanji components) helps you remember meanings and sometimes readings.

2. **Practice stroke order**: Correct stroke order makes writing easier and your kanji more readable.

3. **Use mnemonics**: Create memory aids to remember kanji shapes and meanings.

4. **Study in context**: Learn kanji within words and sentences, not in isolation.

5. **Write regularly**: Physical writing helps reinforce memory.

6. **Group by themes**: Study related kanji together (e.g., all time-related kanji).

7. **Focus on common readings**: Start with the most frequently used readings.

8. **Use flashcards**: Spaced repetition is very effective for kanji learning.

## Practice Quiz

### Quiz 1: Match the reading

1. 日本 → (      )
2. 食べる → (      )
3. 大きい → (      )
4. 学校 → (      )
5. 見る → (      )

**Answers:**
1. にほん
2. たべる
3. おおきい
4. がっこう
5. みる

### Quiz 2: Choose the correct kanji

1. This is a (person) → これは (　) です。 a. 人 b. 入 c. 八
2. I drink (water) → (　) を飲みます。 a. 本 b. 水 c. 木
3. (Mountain) is beautiful → (　) が美しいです。 a. 川 b. 出 c. 山
4. I go to (school) → (　) に行きます。 a. 学校 b. 校学 c. 字学

**Answers:** 1. a 2. b 3. c 4. a

## Conclusion

Mastering these N5 kanji is your foundation for Japanese literacy. Practice regularly, pay attention to both readings, and always learn kanji in the context of actual words and sentences. With consistent practice, these characters will become second nature!
